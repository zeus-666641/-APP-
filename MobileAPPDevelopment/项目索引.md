# 项目索引（AI导航文件）

> 任何AI模型打开此项目时，**首先读这个文件**。它列出所有文件的位置、作用、读取顺序。
> 这是解决"AI无法识别zip"的核心方案：不打包，给地图。

## 四、AI快速上下文加载流程

新会话AI应执行以下步骤：

```
1. Read 项目索引.md（本文件）
2. Read 记忆/项目记忆.md
3. Read 记忆/骨架.md
4. Read 规则.md
5. Read 操作日志.md 的最后20行（了解最近进展）
6. 向用户报告：当前阶段、下一步任务、是否有未完成事项
7. 等待用户确认后开始执行
```

这套流程让任何模型都能在1分钟内进入工作状态。
